# 📎 Клипси — Умный AI-Помощник для Chrome

Расширение-помощник вдохновлённое легендарной «скрепкой» из MS Office, но теперь с настоящим AI (Claude от Anthropic)!

## 🚀 Установка

1. **Распакуйте архив** в удобную папку на компьютере

2. **Откройте Chrome** и перейдите по адресу:
   ```
   chrome://extensions/
   ```

3. **Включите режим разработчика** — переключатель в правом верхнем углу

4. **Нажмите «Загрузить распакованное»** и выберите папку с расширением

5. Значок 📎 появится в панели инструментов Chrome!

## 🔑 Настройка API ключа

1. Получите бесплатный API ключ на [console.anthropic.com](https://console.anthropic.com)
2. Нажмите на иконку 📎 в панели Chrome
3. Введите ключ в поле «API Ключ Anthropic» и нажмите «Сохранить»

Как получить API ключ на Anthropic Console
1. Регистрация
Перейди на console.anthropic.com и создай аккаунт (email или Google).
2. Верификация номера телефона
После регистрации потребуется подтвердить номер телефона.
3. Получение ключа

В левом меню выбери "API Keys"
Нажми "Create Key"
Скопируй и сохрани ключ — он показывается только один раз


Насчёт бесплатного доступа:
Anthropic предоставляет небольшой стартовый кредит новым пользователям (обычно $5), но полностью бесплатного постоянного доступа к API нет — это платный сервис с оплатой по использованию.

## ✨ Возможности

### В попапе расширения:
- 💬 **Чат с AI** — задайте любой вопрос
- 📄 **Кратко о странице** — мгновенный саммари
- 🌍 **Перевести** — перевод содержимого страницы
- ✏️ **Проверить текст** — исправление ошибок
- 🔍 **Объяснить просто** — простое объяснение темы

### Плавающий виджет (на каждой странице):
- 📎 Кнопка в правом нижнем углу — нажмите для открытия чата
- 💬 Выделите любой текст на странице → появятся кнопки:
  - **Объяснить** — объяснение выделенного текста
  - **Перевести** — перевод на русский
  - **Улучшить** — редактирование и улучшение текста
- 🎯 Подсказки появляются автоматически при бездействии

### Контекстное меню:
- Правый клик на выделенном тексте → «Клипси: Объяснить» / «Клипси: Перевести»
